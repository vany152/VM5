Detailed Description

базовый класс команды процессора.

формат команды:
p opcode s r1 r2 offset, где

p - префикс: t - tiny (короткая команда), 2 байта
b - big (длинная команда), 4 байта

opcode - числовой код команды

s - размер операнда: 0 - короткое целое число, 2 байта
1 - вещественное или длинное целое число, 4 байта

r1 - регистр первого операнда
r2 - регистр второго операнда
offset - короткое беззнаковое число, используется и обрабатывается только в длинных командах
